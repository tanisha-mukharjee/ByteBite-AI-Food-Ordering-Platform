## 🚀 Overview

ByteBite is an AI-powered food ordering platform designed to provide a modern restaurant and food delivery experience. The platform combines intelligent food recommendations, secure authentication, online payments, and order management into a single application.

The project focuses on delivering a scalable and user-friendly solution that demonstrates real-world software engineering concepts including backend development, database integration, authentication, payment processing, and AI-assisted customer interactions.

---

## 🌟 Why ByteBite?

Most food ordering platforms focus only on transactions.

ByteBite combines:

* AI-powered recommendations
* Smart chatbot assistance
* Secure authentication
* Online payments
* Personalized user experience

to create a modern food ordering ecosystem rather than just a food delivery website.

The project was developed to demonstrate full-stack development, backend engineering, AI integration, and real-world software architecture concepts.

## ⚙️ Installation Guide

### Prerequisites

* PHP 8+
* MongoDB
* Composer
* XAMPP

### Clone Repository

```bash
git clone https://github.com/tanisha-mukharjee/AI_FOOD_ORDER_SYSTEM.git
```

### Navigate to Project

```bash
cd AI_FOOD_ORDER_SYSTEM
```

### Install Dependencies

```bash
composer install
```

### Configure Database

Update MongoDB connection settings.

### Configure Google OAuth

Create your own Google OAuth credentials and update:

```php
includes/google_config.php
```

### Configure Stripe

Add your Stripe API Keys in:

```php
includes/stripe_config.php
```

### Start Server

Launch Apache and MongoDB.

Open:

```text
http://localhost/AI_FOOD_ORDER_SYSTEM/public
```
